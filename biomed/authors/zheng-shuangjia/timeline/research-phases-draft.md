# 郑双佳 - 研究路线初步分期建议

> **注意**: 本分期为基于领域常识的初步推测，需要阅读具体论文后验证和调整

## 分期框架

### Phase 1: 基础方法建立期 [约 2018-2020]

**时间范围**: 推测为博士阶段或早期独立研究  
**核心特征**: 建立计算药物发现的基础方法论

**可能的研究主题**:
- 传统机器学习方法在药物发现中的应用
- 分子描述符和指纹的开发
- 定量构效关系 (QSAR) 建模
- 基础的数据挖掘和统计学习方法

**预期关键词**:
- Random Forest, SVM
- Molecular fingerprints
- QSAR, QSPR
- Virtual screening

**待验证里程碑**:
- [ ] 第一篇第一作者论文
- [ ] 博士论文主题
- [ ] 早期合作网络建立

---

### Phase 2: 深度学习方法探索期 [约 2020-2022]

**时间范围**: 推测为博士后或独立研究初期  
**核心特征**: 引入深度学习，聚焦表示学习和预测任务

**可能的研究主题**:
- 图神经网络 (GNN) 用于分子表示
- 深度神经网络用于性质预测
- 蛋白质-配体相互作用预测
- 多任务学习和迁移学习

**预期关键词**:
- Graph Neural Network (GNN)
- Deep learning
- Molecular representation
- Binding affinity prediction
- Multi-task learning

**方法学创新点** (推测):
- 提出新的 GNN 架构用于分子
- 开发多模态分子表示方法
- 改进蛋白质-配体预测精度

**待验证里程碑**:
- [ ] GNN 相关代表作
- [ ] 高引用 (>50) 论文出现
- [ ] 方法被后续工作采用

---

### Phase 3: 生成模型与系统整合期 [约 2022-2024]

**时间范围**: 近期活跃期  
**核心特征**: 从预测转向生成，构建完整药物设计系统

**可能的研究主题**:
- 分子生成模型 (VAE, GAN, Flow-based)
- 从头药物设计 (De Novo Design)
- 逆合成规划 (Retrosynthesis)
- 多目标优化 (Multi-objective Optimization)
- 先导化合物优化

**预期关键词**:
- Generative models
- VAE, GAN, Flow, Diffusion
- De novo design
- Lead optimization
- Retrosynthesis
- Multi-objective optimization

**系统级贡献** (推测):
- 构建端到端药物设计平台
- 整合多种计算方法
- 面向实际应用的方法开发

**待验证里程碑**:
- [ ] 生成模型代表作
- [ ] 开源工具/平台发布
- [ ] 工业界合作或应用

---

### Phase 4: 前沿拓展与大模型期 [约 2024-至今]

**时间范围**: 当前及未来  
**核心特征**: 拥抱大模型时代，探索 Foundation Models

**可能的研究主题**:
- 生物分子基础模型 (Foundation Models)
- 大规模预训练 + 下游微调
- 扩散模型 (Diffusion Models) 用于 3D 分子
- 多模态融合 (序列 + 结构 + 文本)
- 酶工程与合成生物学

**预期关键词**:
- Foundation models
- Large language models
- Pre-training
- Diffusion models
- 3D molecule generation
- Multi-modal learning
- Enzyme engineering

**前沿方向** (推测):
- 类似 AlphaFold 的结构预测方法
- ChatGPT-like 的药物发现助手
- 跨模态知识整合

**待观察趋势**:
- [ ] 是否有 LLM 相关工作
- [ ] 是否涉及 Diffusion Models
- [ ] 是否有 3D 生成工作
- [ ] 国际合作扩展
